LibreSpeed 魔改 Docker 版

https://hub.docker.com/r/jaydenlee2019/speedtest

简单说明：

1、基于 php:fpm-alpine3.12 镜像；

2、开启 php opcache，设置 cgi.fix_pathinfo=0， date.timezone=Asia/Shanghai；

3、集成 nginx 1.8.0；

4、美化 UI，黑色主题；

5、IP定位使用B站API。

运行示例：

docker run --name=speedtest -p 8081:80 -d jaydenlee2019/speedtest

自定义Title：

docker run --name=speedtest -p 8081:80 -e TITLE="iON Cloud 圣何塞测速站点（非官方）" -d jaydenlee2019/speedtest

更新说明：

1.1.0版本

1、减少镜像的层数；

2、增加自定义Title
